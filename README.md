# Projeto 3D-DevArt

Projeto de portfólio moderno com foco em experiências digitais imersivas, combinando animações avançadas, renderização 3D e uma interface responsiva. Desenvolvido através do DevArt - Agência 3D do Desenvolvedor Gustavo Campelo, ao qual é o autor principal do projeto 3D.

---

## ✨ Sobre o projeto

Este projeto foi desenvolvido para apresentar serviços e projetos de forma visualmente impactante, utilizando animações baseadas em scroll e elementos 3D com Three.js.

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* GSAP (ScrollTrigger, ScrollSmoother, SplitText)
* Three.js
* GLTFLoader

---

## 🎬 Funcionalidades

* 🔥 Animações sincronizadas com scroll
* 💎 Renderização de objeto 3D (diamante)
* 🎨 Efeito de saturação dinâmica nas imagens
* 📱 Layout responsivo (desktop, tablet e mobile)
* 🎥 Vídeos como background
* ⚡ Transições suaves entre seções

## ⚙️ Como rodar o projeto

> ⚠️ Necessário rodar em servidor local (por causa do Three.js e módulos ES6)

### Usando VS Code

1. Instale a extensão **Live Server**
2. Clique com botão direito no `index.html`
3. Selecione **"Open with Live Server"**

---

## 📱 Responsividade

O projeto foi adaptado para diferentes dispositivos com:

* Uso de `clamp()` para tipografia
* Layout flexível com Flexbox
* Ajustes de animações via GSAP (`matchMedia`)
